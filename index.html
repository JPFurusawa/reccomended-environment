推奨環境

当サイトでは下記の環境で快適にご利用いただけます。

1. ご利用のブラウザやデバイス

Windows PC

Windows10にていずれかのブラウザで閲覧

Mac OS

Appleのサポート対象となっているOSにて Google Chromeで閲覧

iPhone

iOS13以上にてSafariで閲覧

Android

Android OS 7以上 にて Google Chrome で閲覧

  • 各ブラウザ・OSは最新バージョンを推奨します。
  • ただし上記環境範囲内であっても下記条件下においては、一部表示不具合や各種機能がご利用になれない場合があります。
  • ・プライベートブラウジングモードの利用
  • ・アドブロックなど各種アドオンの利用
  • ・各種ブラウザのベータ版やプレビュー版を使用している場合
  • ・一部のブラウザのバージョンとOSのバージョンの組み合わせの場合
  • ・2つ以上のタブを開きながらそれぞれのタブで、同時にお見積もりや契約などのお手続きを行う場合

2. セキュリティ

当サイトでは個人情報を保護し安心してご利用いただくために、HTTPS（保護された接続）による通信を使用します。

3. PDFファイルの閲覧

PDFファイルを閲覧・印刷する場合にはAdobe Readerが必要となります。

4. JavaScript

JavaScriptを使用しているコンテンツがあります。ご利用のブラウザでJavaScriptの機能を無効にされている場合は、一部のコンテンツが正しく表示されない、もしくは正しく機能しないことがあります。ブラウザの設定でJavaScriptを有効にしてください。

5. 二次元コード（QRコードなど）

各ウェブページに記載の二次元コードはスマートフォンなどのカメラで読み取ることで利用可能です。
※ QRコードは（株）デンソーウェーブの登録商標です。

6. Cookieなどの利用

チューリッヒ保険会社では、ウェブサイトの利用状況に関する統計分析、また、お客さまにより良い情報、サービスや広告を提供していくことを目的として、Cookieなどを利用して一定の情報を収集する場合があります。

Cookieとは、お客さまがウェブサイトを訪れた際に、ウェブサーバーからお客さまのコンピューターやモバイルデバイス内に記録される小さなテキストファイルのことで、それによりコンピューターやモバイルデバイスを識別することができるものです。
